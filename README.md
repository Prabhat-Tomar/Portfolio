# Prabhat Tomar — Portfolio v2 (Grey Theme + Dark Mode)

Minimalist grey portfolio with dark mode, bold typography, scroll animations, and tool logos.

## 📁 Files

```
portfolio-v2/
├── index.html    # Full portfolio
├── style.css     # All styles + dark mode + responsive
├── script.js     # Dark mode, scroll curtain, reveal, nav
├── profile.jpg   # Add your photo here (same folder)
└── README.md
```

## 🖼️ Add Your Photo
Place `profile.jpg` in the same folder as `index.html`.

## 🌐 GitHub Pages
1. Create a new repo on GitHub
2. Upload all files (including `profile.jpg`)
3. Settings → Pages → Source: main → Save
4. Live at `https://[username].github.io/[repo]/`

## ✨ Features
- Dark / Light mode toggle (saved to localStorage)
- Scroll curtain wipe animation between sections
- Scroll-triggered fade-in reveals
- Bordered profile photo with double ring
- Tool logos (Power BI, MySQL, Python, Excel, Canva)
- Bold highlighted skill names
- Subtle grid background pattern in hero
- Dot & line background patterns
- Fully responsive (mobile, tablet, desktop)
- Compact footer with clear typography
